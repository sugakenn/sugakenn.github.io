# 原則

マークダウンでは基本的に、文字コードはUTF-8、改行はLFで記述します。

マークダウン専用の記述式だけでなく、多くのHTMLタグが使えます。ただし、HTMLタグの内側ではマークダウンの書式は使えません。

---

# 見出し

`#` の数で見出しのレベルを指定します。

## 書き方

```md
# 見出し1
## 見出し2
### 見出し3
#### 見出し4
```

---

# 段落

空行を入れると新しい段落になります。

## 書き方

```md
これは1つ目の段落です。

これは2つ目の段落です。
```

---

# 改行

Markdownでは改行だけでは改行されません。

## 行末にスペース2つ

```md
1行目␠␠
2行目
```

※「␠」はスペースを表します。

または

```md
1行目<br>
2行目
```

---

# 強調

## 太字

```md
**重要**
```

## 斜体

```md
*重要*
```

## 太字＋斜体

```md
***重要***
```

## 取り消し線

```md
~~削除~~
```

# インラインコード

文章中にコードを書きたい場合はバッククォートで囲みます。

```md
`printf()`
```

複数行のコードはバッククォート3つで囲みます。

````md
```c

```


```python

```c
```php

```json
````

コードブロックのバッククォートは3つ以上も設定でき、数が多い場合はその数以上のバッククオートが表れれない限りコードブロックが終了しません。

# テーブル

テーブルは | を使って表します。テーブル内でもマークダウンを


```md
| タイトル    |          |   |
| ----------- | -------- | - |
| 内容        |          |  |

```
---はヘッダとデータの区分けにつかいますが、ここで

:--- とすれば左寄せ

---: とすれば右寄せ

:----:とすれば中央寄せ

となります。

---

# エスケープ

\# や　* などをエスケープしたい場合は \ をつけます。気を付けないといけないのは不要なエスケープをすると\はそのまま文字列として扱われてしまうところです。

またhtmlタグを例示したいような場合は\`で囲んでコードブロックにすることでエスケープが可能です。  
同様に
````md
```
````
で囲まれた中なら、エスケープは不要になります。

コメントはマークダウン記法としては存在しませんが、HTMLのタグ`<!-- -->` が使えます。
